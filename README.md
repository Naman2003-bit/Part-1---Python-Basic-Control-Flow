# Part 1 - Student Grade Tracker

## Description
A command-line program that cleans raw student data,
analyzes marks, and generates performance reports.

## Tasks
- Task 1: Data Parsing & Profile Cleaning
- Task 2: Marks Analysis using Loops & Conditionals
- Task 3: Class Performance Summary
- Task 4: String Manipulation Utility

## How to Run
```bash
jupyter notebook part1_grade_tracker.ipynb
```

## Concepts Used
Python basics, loops, conditionals, f-strings, string methods
